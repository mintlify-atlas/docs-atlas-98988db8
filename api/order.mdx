## Overview

The `Order` class is the core domain model that represents an e-commerce order in the system. It maintains the order's unique identifier and current status throughout its lifecycle.

## Fields

<ResponseField name="id" type="String">
  Unique identifier for the order
</ResponseField>

<ResponseField name="status" type="String">
  Current status of the order. Common values include:
  - `"CREADO"` - Order has been created
  - `"ENVIADO"` - Order has been shipped
</ResponseField>

